# PulseFi Frontend

The lender experience frontend for PulseFi - The Trust Engine for SME Lending.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

- `/app` - Next.js 13+ app directory with pages and layouts
- `/components` - Reusable React components
- `/lib` - API client and utilities
- `/types` - TypeScript type definitions

## Key Features

- Lender authentication and registration
- SME marketplace with filtering
- Detailed SME profiles with Pulse and Profit scores
- Responsive design with Tailwind CSS
